# Netflix Landing Page Clone
A clone of netflix landing page, static + interactable 

📖 **About the Project**
This is a netflix landing page clone which is close to the original one. 
The website is static with micro interactions featuring smooth transitions between elements.
This project is a part of my learning to becoming a full stack developer.

🛠️**Tech Stack**
HTML
CSS
JS
GitHub
VS Code

🧠 **What I Learned**
With this project I learned how websites divide their website into different sections which futher sub divide, which actually
makes it easier to work on small elements which than all come together and become one good website. Micro interactions and making
the buttons functioning in the trending section allowed me to step into JavaScript officially.
